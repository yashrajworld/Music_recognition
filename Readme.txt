To test this project
''''''''''''''''''''''
1. Required python libraries : 
	a. tensorflow
	b. librosa
	c. numpy
	d. pandas
	e. os

2. Required dataset for training : 
	GTZAN dataset: "genres" folder in same directory as the python notebook.

3. Run "music recog.ipynb" notebook to train and test dataset.

4. Example file "disco.00001.wav" in same directory as the python notebook.

5. "songs" folder has a song file whose genre has been predicted later in the notebook.
